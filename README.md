## 🌍 Philosophy & Vision / 开发理念与愿景

I believe great systems are:  
我相信优秀的系统应该：

- **Intuitive in behavior**, yet **deep in interaction**  
  行为直觉清晰，交互深度丰富

- **Rule-bound, not script-bound**  
  受规则驱动，而非剧本驱动

- **Built for emergence**, not mere execution  
  支持涌现，而非单一执行路径
